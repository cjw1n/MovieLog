# 🎬 CineLog: 나만의 영화 아카이브 매뉴얼

CineLog는 영화광들을 위한 개인 감상평 기록 도구입니다. 영화 정보를 수동으로 입력할 필요 없이 검색을 통해 빠르게 기록할 수 있습니다.

## ✨ 주요 기능
- **영화 검색 시뮬레이션**: 제목 입력 후 검색 버튼을 누르면 DB에서 정보를 불러옵니다. (현재 인기 영화 10종 내장)
- **카드형 UI**: 기록된 영화들을 깔끔한 카드 형태로 한눈에 확인할 수 있습니다.
- **별점 및 한 줄 평**: 시각적인 별점과 함께 짧은 감상평을 영구적으로 보관합니다.
- **데이터 자동 저장**: 모든 기록은 `my_movies.json` 파일에 안전하게 저장됩니다.

## 🛠 설치 및 실행 방법
1. `CineLog.exe` 실행
2. `customtkinter` 라이브러리 필요 (소스 실행 시)
