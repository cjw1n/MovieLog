# 영화 관람 기록

영화광들을 위한 개인 감상평 기록 도구

## 기능
- **영화 검색 시뮬레이션**: 제목 입력 후 검색 버튼을 누르면 DB에서 정보를 불러옴
- **카드형 UI**: 기록된 영화들을 깔끔한 카드 형태로 한눈에 확인
- **별점 및 한 줄 평**: 시각적인 별점과 함께 짧은 감상평을 영구적으로 보관
- **데이터 자동 저장**: 모든 기록은 `my_movies.json` 파일에 안전하게 저장됨

## 설치 및 실행
```
CineLog.exe 실행
customtkinter 라이브러리 필요 (소스 실행 시)
```
